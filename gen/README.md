# Commands for generating files

## > cleanup
### `output`
> Is removed.

## > item_model
### `input`
> Contains item textures.
### `ouput`
> Generates minecraft:item/generated file and moves `input` to correct path.

## > loot_table
### `input`
> Contains blockstates.
### `ouput`
> Generates loot table file and moves `input` to correct path.
